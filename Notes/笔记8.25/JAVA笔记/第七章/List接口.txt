


list是 collection的子接口 
可以重复 且有顺序

List容器中的元素都对应一个整数型的序号记载其位置。

List包涵  ArrayList LinkedList 


indexOf 求出元素所在的位置
latIndexOf 求出元素所在的最后一个位置

List 常用算法 ：
	Void sort(List) 对List容器内的元素排序
	void shuffle(List) 对List 容器内的对象进行随机排序
	void reverse（List) 对list容器内的对象进行逆向排列 注意 只能单纯逆序排列 不会先进行正序排列 再逆序排列
	void fill(List,Object) 用一个特定的对象重写整个List容器
	void copy(List dest ,List src) 将Src List容器内容拷贝到 Dest list 容器之中去
	int binarySearch(List ,Object)对于顺序的list容器 采用二分法的 查找内容方法
