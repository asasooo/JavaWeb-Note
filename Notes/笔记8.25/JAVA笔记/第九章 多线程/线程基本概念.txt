线程是程序中不同的运行路径 一般是多个执行路径

线程 可以并行 运行

main方法是 主进程 
  进程 的 意思是 ：机器上的 exe文件 将程序的代码放进代码区 是一个静态的概念 一般指main方法开始进行

机器同时进行多个线程 进程
 

每一个Thread 代表一个新的线程


	java的线程是通过JAVALANGThread来实现
	vm启动时会创建一个 主方法  所定义的线程
	可以通过创建 Thread 的实例创建的线程
	每个线程 都是通过某个特定的thread 对象所应用的方法 run（）来完成其操作的 方法run 成为线程体
	通过调用thread 泪的start 方法 来启动一 线程

一下方法 可以二选一 ：
  1. 接口 runnable
	run（）


  2.  继承  Thread(Runnable target)
	通过 start 方法 调用 


可以通过 InterruptedException 用 catch 然后做出 return 来结束线程

或者 先定义一个 boolean while flag = true  用改变flag 来停止线程
