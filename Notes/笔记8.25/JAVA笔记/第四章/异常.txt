java异常 通常指运行期出现的错误。

JAVA程序的执行过程中出现异常 可以生成一个异常类 并提交给java
此过程称为 throw 抛出。
try catch 可以更改 错误提示

示例代码：
     public class TryCatch {
	public static void main(String arg[]) {
	int[] a = {1,2,3};
	try {
	System.out.print(a[3]);
	}catch(ArrayIndexOutOfBoundsException e) {
		System.out.println("Out Of Range!");
		e.printStackTrace();
		}
	}
}
 	打印出 java原本应显示的错误提示。
	其中 e 只是一个形参 叫什么都可以 只有在出现错误时 才会调出形参。


e.printStackTrace() 方法 可以打印出本应该打印出的错误提示。

一个try可以跟随多个catch  根据从小到大级别

try之后的catch写可能throw的错误。

try方法的finally中的代码无论如何 都会执行
通常在finally语句中都会进行清理垃圾。

catch异常之后 一定要做处理。若不处理就会看不见异常。

在catch 时 应该先catch小异常 在catch大异常。

重写需要抛出与原方法抛出异常类型一致的异常或者不抛出异常。方法
