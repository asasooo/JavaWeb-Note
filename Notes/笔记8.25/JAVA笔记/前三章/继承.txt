OverWrite  必须与重写前的方法有相同的 方法名 返回类型 声明。
  重写只限于 子类 中重写父类的 方法。
重写的方法不能比被重写方法 更加严格的访问权限。

子类继承父类，子类拥有了父类的所有属性和方法。private也可以继承。但是private 只能看不能用。

this是当前对象的方法的引用 super是父类对象方法的引用

继承之中的 构造方法 :子类的构造过程之中必须调用父类的构造方法。
继承之中可以在子类的构造方法之中 调用 父类的构造方法。 若要用super调用父类的构造方法，则一定要在子类构造方法第一行进行调用。
调用子类的构造方法时 必须 调用父类的构造方法，若子类的构造方法有参数 ， 但父类方法之中没有构造方法含有相同参数 则 调用父类方法之中没有参数的构造方法。


====================================


当子类的构造方法没有调用父类的构造方法 即没有写super时 系统默认去载入父类之中没有参数的构造方法。若此时父类之中没有 没有参数的构造方法 则会报错。
在子类之中可以用 this 来调用子类里的构造方法。

参数列表：参数列表中的参数用逗号分开，列表中包含了传递给调用函数的变量的声明。如果函数不接收任何参数，参数列表就声明为void。

函数定义的一般格式为：
	权限类型  返回值类型 函数名 （参数列表）
	{
		声明
		语句
	}

object 是所有java类的父类
